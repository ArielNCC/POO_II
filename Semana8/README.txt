# Para correr el programa.

## Paso 1.
Correr el archivo docker-compose.yml en terminal
docker-compose up -d
-- Considerar que el puerto de conexión sera el 3307
-- Usuario y Password en el archivo Exp3/CineMagenta/db/db.properties

## Paso 2.
Crear una conexión en IDE MySQL (sugiero Workbench)

## Paso 3.
Ejecutar el script mysql Query_magenta.sql con usuario administrador o con los permisos necesarios.

## Paso 4. 
Abrir el proyecto java CineMagenta en el IDE.
